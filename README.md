# Bootcamp Desenvolvimento e Projeto de Sistemas Modernos - TaskFlow: Gerenciamento de Tarefas em Equipe

Bem-vindo ao projeto **"TaskFlow"**, onde este projeto está sendo desenvolvido no âmbito do Bootcamp de Desenvolvimento e Projeto de Sistemas Modernos, com a proposta de aplicar na prática os conhecimentos adquiridos em programação orientada a objetos (back-end em Java), estruturação e estilização de interface web (front-end) e versionamento de código.
A proposta é criar um **sistema web para gestão de tarefas de uma pequena equipe**, trazendo uma maior organização, transparência e produtividade para o dia a dia de equipes.

## Introdução

O projeto tem como foco resolver o problema do **descontrole de tarefas em equipes**. Sem uma ferramenta de organização, as equipes perdem o controle de suas metas, atividades a serem entregues, prazos de envio e a falta de visualizar o que cada membro da equipe está fazendo.

Atualmente, o gerenciamento manual de tarefas apresenta limitações como:

- Falta de clareza do que cada membro está executando;
- Falta/Pouca acessibilidade;
- Ausência de relatórios claros sobre o progresso das atividades;
- Dificuldade no controle de prazos e na produtividade da equipe.

Diante disso, propomos o desenvolvimento de um **TaskFlow**, um sistema web responsivo para gerenciamento de tarefas, com o papel fundamental em oferecer uma navegação objetiva, permitindo a cadastro de usuários, criação e atribuição de tarefas, controle de status (A Fazer, Em Andamento e Concluída), filtros e relatórios automáticos com percentual de conclusão.

## Ideação

A ideia principal do nosso projeto é oferecer uma solução **intuitiva, organizada e objetiva** para o gerenciamento colaborativo de tarefas.

O sistema apresentará:

- Uma experiência agradável e interativa ao usuário;
- Cadastro de usuários e tarefas;
- Organização clara de tarefas por status, responsável e prioridade;
- Design responsivo, atrativo e de fácil navegação;
- Compatibilidade com diferentes plataformas (desktop e mobile).

## Prototipação, Tecnologias e Funcionamento

A prototipação do projeto está sendo realizada em um **Ambiente de Desenvolvimento Integrado (IDE)**, utilizando principalmente o **Visual Studio Code (VSCode) e o Intellij.**

As tecnologias previstas para o desenvolvimento, funções e etapas serão:

### Front-End

- HTML5 -> Estruturação do projeto;
- CSS3 + Bootstrap -> Estilização e responsividade;
- JavaScript -> Interatividade para o sistema.

### Back-End

- Java com Programação Orientada a Objetos **(POO);**

### Ferramentas

- Git/GitHub -> Versionamento e colaboração em grupo;
- VSCode -> IDE para o desenvolvimento do front-end;
- Intellij IDEA -> IDE para o desenvolvimento do back-end.
