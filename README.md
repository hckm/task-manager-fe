# Bootcamp: Desenvolvimento e Projeto de Sistemas Modernos - TaskFlow: Gerenciamento de Tarefas em Equipe

Bem-vindo ao **TaskFlow**, o sistema web completo para gestão de tarefas de equipes desenvolvido durante o Bootcamp de Desenvolvimento e Projeto de Sistemas Modernos da Unisagrado.
O objetivo foi aplicar na prática todos os conceitos de **Programação Orientada a Objetos (POO)** no back-end Java, estruturação e estilização de interface web responsiva no front-end e versionamento colaborativo com Git/GitHub.

Versão Online (Deploy): https://hckm.github.io/task-manager-fe/

## Visão Geral

O TaskFlow resolve o clássico problema do **descontrole de tarefas em equipes**:

- Falta de clareza do que cada membro está executando;
- Ausência de relatórios claros de progresso;
- Pouca acessibilidade e transparência;
- Dificuldade no controle de prazos.

Com o TaskFlow, a equipe ganha um melhor gerenciamento, obtendo **organização, transparência, produtividade e modernidade**.

---

## Funcionalidades Implementadas

- Cadastro e gerenciamento completo de **Responsáveis** (com cargos);
- Cadastro e gerenciamento de **Cargos**;
- Criação de tarefas com título, descrição, prioridade, prazo e responsável;
- Controle de status (A Fazer → Em Andamento → Concluído);
- Dashboard com métricas em tempo real (Total, A Fazer, Em Andamento, Concluídas);
- Filtros avançados por responsável, status e período (data);
- Paginação inteligente das tarefas;
- Alteração de status diretamente nos cards com salvamento no back-end;
- Exclusão de tarefas
- Design totalmente responsivo (desktop e mobile);
- Integração completa **Front-end + Back-end** via API REST;
- Arquitetura com **Programação Orientada a Objetos (POO)**.

---

## Tecnologias Utilizadas

### Back-End (Java + POO)

- **Java** com forte aplicação de Programação Orientada a Objetos;
- Entidades, Services, Repositories e Controllers;
- Encapsulamento, Herança, Polimorfismo e Abstração;
- Spring Boot (API REST).

### Front-End

- **HTML5** -> Estruturação;
- **CSS3 + Bootstrap 5** -> Estilização e responsividade;
- **JavaScript** -> Interatividade.

## Ferramentas

- **Git & GitHub** -> Versionamento e colaboração;
- **VSCode** -> Desenvolvimento do Front-End;
- **Intellij IDEA** -> Desenvolvimento do Back-End.

## Conclusão

O **TaskFlow** foi entregue totalmente integrado (Front-End + Back-End), moderno, responsivo e funcional. O projeto cumpre todos os requisitos do bootcamp, demonstrando domínio de:

- Programação Orientada a Objetos (POO);
- Consumo de API REST;

Desse modo, o TaskFlow não é apenas um gerenciador de tarefas. é uma solução real que aumenta a produtividade e a transparência de qualquer equipe
